Course Description

The official syllabus can be found here. This course covers the topics typically covered in the standard Calc 1 and Calc 2 courses. This is a VERY fast paced course as it covers two semesters worth of material in one and you also learn to write mathematical proofs. If you like being challenged and are curious and excited about maths this is the course for you. This course will be taught in the mode of Inquiry Based Learning (see below).

Homeworks

Homeworks are what make this course. There will be reading assignments to be read BEFORE every class. I'll hand out problem sets in each class which you'll discuss in small groups in class, write the solutions by yourselves and submit them on the following Monday. Depending on how far we get with the problem sets during the week each group will get a chance to present their solutions to the rest of the class.

Grading Scheme

Class participation : 30%
Homeworks : 40%
Final Exam : 30%

Each week's problem sets count as one homework, only the top 10 homeworks will be included in the final grade.

IBL

“The only way to learn mathematics is to do mathematics.”

- Paul Halmos

In an IBL course there will hardly be any lectures to prove things for you. Rather, you will prove the propositions yourself and explain your proofs to each other. The IBL format fosters creativity and a deep understanding of the material. However, its success depends heavily on the students: you must be actively engaged and prepared for class every day.
