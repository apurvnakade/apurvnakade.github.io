Curvature of Curves
Filed under Gaussian Curvature, April 8, 2018.

If we agree that lines have no curvature and circles have constant curvature then we’re more or less forced to say that curvature of a circle should be inversely proportional to the radius, after all a line is a circle with an infinite radius. We might as well

Define the curvature \(\kappa\) at every point of a circle of radius \(r\) to be \(1/r\) .

For more general curves, for example, a sine curve, different points should have different curvatures. As we know what the curvature of a circle is, the standard trick in maths is to take an arbitrary curve \(C\) and at each point \(p \in C\) try to find the circle that approximates it the best, the best fitting circle (apparently called an osculating circle). But how to find such a circle? And how would we know we’ve found the best fit?

Enter Calculus.

Curvature is Second Order

A circle of radius \(r\) can be described by $$x^2 + y^2 = r^2$$

This is a quadratic expression which suggests that curvature is a quadratic phenomenon, in much the same way as slope is a linear phenomenon. In calculus, the Taylor series relates quadratic behavior to second derivatives, so let’s find second derivatives.

$$\begin{align} x^2 + y^2 &= r^2 \\ 2x + 2yy' &= 0 \\ 2 + (2y'^2 + 2yy'') &= 0 \end{align}$$

Nothing that interesting. But at the two critical points on the circle \((0,r)\) and \((0,-r)\) where \(y'=0\) this equation simplifies to $$\begin{align} |y''| &= 1/r \end{align}$$ But this is exactly the curvature!! So we’ll make a leap and define,

Definition. For a graph \(C\) of \(f(x)\) if \(f'(x_0)=0\) then $$\kappa = |f''(x_0)|$$

is the curvature of \(C\) at \((x_0, f(x_0)).\)

For example, if \(y=x^2\) we can see that the circle of radius \(1/y''(0) = 0.5\) best fits the parabola at \((0,0)\)

The conditions imposed on \(C,\) \(f\) and \(x_0\) are not too stringent. By rotating the ambient \(\mathbb{R}^2\) (which should not change the curvature) we can arrange for any \(C\) to be in the above form and so this defines curvature for arbitrary curves!

Curvature as Acceleration

There is a quick way to find curvature using ideas from physics, after all the second derivative is related to acceleration.

We can parametrize a circle of radius as $$\begin{align} c(t) = (r \cos (\omega t), r \sin (\omega t)) \end{align}$$ If we assume that \(\omega\) is a constant then we get $$\begin{align} c'(t) &= (-r \omega \sin (\omega t), r \omega \cos (\omega t)) \\ c''(t) &= (-r \omega^2 \cos (\omega t), -r \omega^2 \sin (\omega t)) \end{align}$$ so that $$\begin{align} \kappa = \dfrac{1}{r} = \dfrac{||c''(t)||}{||c'(t)||^2} \end{align}$$ And so we make the second leap.

Definition. For a graph \(C\) parametrized as \(c(t)\) with constant speed \(||c'(t)||\) the curvature is given by $$\begin{align} \kappa(t) = \dfrac{||c''(t)||}{||c'(t)||^2} \end{align}$$

Unfortunately, the condition of \(||c'(t)||\) being constant is not easy to impose. So we need a formula for the case when \(||c'(t)||\) is varying, which in the case of circle translates to saying that \(\omega\) is a function of \(t\) . The details in this case are a bit gory but the final answer is quite nice.

Proposition. For a graph \(C\) parametrized as \(c(t)\) the curvature is given by $$\begin{align} \kappa(t) &= \dfrac{||c'(t) \times c''(t)||}{||c'(t)||^3} \\ &= \dfrac{\mbox{normal component of acceleration}}{\mbox{speed}^3} \end{align}$$
