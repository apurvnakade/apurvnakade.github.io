Course Info

“The only way to learn mathematics is to do mathematics.”

- Paul Halmos

The textbook for the class is Calculus, 3rd Edition, Spivak. We'll only use this book as a reference. The official syllabus can be found here.

This class will be structured in the mode of Inquiry Based Learning. In this class there will hardly be any lectures to prove things for you. Instead, you will prove the propositions yourself and explain your proofs to each other. The environment of an IBL classroom is collaborative and not competitive and the main goal is to have loads of fun while learning some interesting mathematics.

The first half of the class with be very theoretical and focused on writing proofs, the second half will focus on more standard calculus techniques (while still doing some proofs). Following is a tentative list of topics to be covered:

Continuity How to prove continuity/discontinuity of a function using the ε-δ definition, Intermediate value theorem, inf, sup etc.
Derivatives Basic computations, Mean value theorem, Critical points, min/max problems, Computing Taylor series, estimating errors
Integrals Computations (substitution tricks, integration by parts, trig subs etc.), Fundamental theorem of Calculus
Sequences Basic theorems about convergence
Series Convergence tests for series

Grading Scheme
Class participation : 40%
Homeworks : 40%
Final Exam : 20%

Class participation: This is the largest chunk of your grade, hence attendance is compulsory. If you have a legitimate reason for not being able to attend a class, for eg. if you have caught the flu or if you need to go to Sweden to receive the Nobel prize, then you should let me know asap about your absence, preferably before the class. During the class:

  • You'll be expected to work in groups with others and solve that day's problems (as much as possible).
  • You'll present your solutions to the rest of the class (we'll work on this).

Homeworks: Homework will consist of two parts:

  • Each class you'll be handed a problem sheet to work on together, however, you're expected to write the solutions by yourselves. Written solutions will be due at the beginning of class on Monday.
  • There will be reading assignments for the weekend. If you do not do the reading assignments you won't be able to do the problems in class and this will affect your class participation grade.
Each week's problem sets count as one homework, only the top 10 homeworks will be included in the final grade (so no late submissions).

Final Exam: We'll cross this bridge when we get to it.

Optional Project
You have an option of doing a reading project in this class. If you choose to do a project, I'll reduce the weight of the final exam for you. You are free to come up with your own topics. For the project you'll have to submit a written report at the end of the semester. The report should have a rigorous definition-theorem-proof structure.
