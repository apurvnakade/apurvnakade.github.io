AS.110.113 - Honors Single Variable Calculus

Updates
The textbook for the class is Calculus, 3rd Edition, Spivak. We'll only use this book as a reference.
The official syllabus can be found here. This class will be taught in the mode of Inquiry Based Learning.

Topics to be Covered
Continuity How to prove continuity/discontinuity of a function using the ε-δ definition,
Intermediate value theorem, inf, sup etc.
Derivatives Basic computations, Mean value theorem
Critical points, min/max problems
Computing Taylor series, estimating errors
Integrals Computations (substitution tricks, integration by parts, trig subs etc.)
Fundamental theorem of Calculus
Sequences Basic theorems about convergence
Series Convergence tests for series

Inquiry Based Learning

“The only way to learn mathematics is to do mathematics.”

- Paul Halmos

In this (IBL) class there will hardly be any lectures to prove things for you. Instead, you will prove the propositions yourself and explain your proofs to each other. The environment of an IBL classroom is collaborative and not competitive and the main goal is to have loads of fun while learning some amazing mathematics.

We will meet during regular class hours on Monday and Wednesday when you'll be given problem sheets to solve. You are expected to form groups and solve the problems together, but write the solutions by yourselves. These solutions and some extra HW problems will be due on Friday when I'll be holding extended office hours.

Grading Scheme
Class participation : 40%
Homeworks : 40%
Final Exam : 20%

Each week's problem sets count as one homework, only the top 10 homeworks will be included in the final grade (so no late submissions).

Optional Project
You have an option of doing a reading project in this class. If you choose to do a project, I'll reduce the weight of the final exam for you. You are free to come up with your own topics. For the project you'll have to submit a written report at the end of the semester. The report should have a rigorous definition-theorem-proof structure.
